subscriptions
service in module sqd

Description

This is a service that is returned as part of subscriptionsProvider. Used to deal with creating a subscription or checking whether a user has subscribed through Google Play or the App Store.

Usage

subscriptions($q, stores, exceptions);

Parameters

ParamTypeDetails
$qobject

Angular promise service

storesobject

Service used to interface with the html5 local storage

exceptionsobject

Service to catch a failed promise

Returns

object

Service object exposing methods - create, read

Methods

  • create(username)

    Sends user to Google Play or App store. Once they confirm a subscription the user is returned to Squidler where the subscirption information is sent to Squidler back-end to be stored.

    ParamTypeDetails

    Returns
    promise
