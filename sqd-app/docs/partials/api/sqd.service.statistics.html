statistics
service in module sqd

Description

This is a service that is returned as part of statisticsProvider. Used to retrieve a statistics information on a particular Squidle

Usage

statistics($q, resources, stores, profiles);

Parameters

ParamTypeDetails
$qobject

Angular promise service

resourcesobject

Service to communicate with backend resources

storesobject

Service used to interface with the html5 local storage

profilesobject

Retrives the public profile for a specific user

Returns

object

Service object exposing methods - read

Methods

  • read(short)

    Retrieves the Squidle stats

    Parameters
    ParamTypeDetails
    shortstring

    Short link id of the squidle

    Returns
    promise

    Resolves to an object with keys 'summary' (summary stats on squidle) and 'players' (an array with player specific stats)

    statistics.read('F36SAb6')
