fullScreenText
directive in module sqd

Description

Creates an interface to edit text in full screen mode

Usage

as element:
<full-screen-text
       id="{string}"
       hide="{function}"
       active="{bool}"
       data="{object/string}"
       placeholder="{string}"
       form="{object}"
       links="{bool}"
       sqd-required="{string}"
       valid="{bool}">
</full-screen-text>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
idstring

Unique id for this full screen text element

hidefunction

reference to a function on the parents scope to be called to hide the modal view

activebool

A boolean to determine if this full screen text view is currently active

dataobject/string

Will hold all the data associated with the input, i.e. text any media. Use object if you want to use fullScreenText with check-for-links directive, otherwise you can just use a string

placeholderstring

Placeholder for the textarea

formobject

Form element that this fullScreenText belongs to

linksbool

If true the content of the fullScreenText is checked to see if it contains links, the content of which is then attached to the media key of the data object (obviously data must be an object and not a string in this case)

sqd-requiredstring

Specifies if the textarea is a required part of the form (if indeed this fullScreenText is part of a form)

validbool

Is the fullScreenText valid or not
