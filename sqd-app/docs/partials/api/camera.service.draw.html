draw
service in module camera

Description

This is a service used to draw on an kineticjs image

Usage

draw($rootScope,, Kinetic);

Parameters

ParamTypeDetails
$rootScope,object

Angular rootScope service

Kineticobject

Kinetic.js wrapped in angular service

Returns

object

Service object exposing methods - start, stop, undo

Methods

  • start(stage, layers, objects, colour)

    Creates the events listeners that watch for user interaction with the image then draw those onto the image

    Parameters
    ParamTypeDetails
    stageobject

    Kinetic.js staage object where the image layer is

    layersobject

    Contains layers that have kinetic.js objects on them, a key named "drawing" must be present whose value must be a kinetic.js layer

    objectsobject

    Contains objects that have been drawn onto the various layers defined in the layers object, a key named "lines" must be present whose value must be an array of kinetic.js lines

    colourobject

    The key of this object named "value" must be the colour of the line that is to be drawn

  • stop(objects)

    Removes the event listeners from a kineticjs image so that the imge no longer responds to touch/click. Essentially this turns drawing mode off

    Parameters
    ParamTypeDetails
    objectsobject

    Contains objects that have been drawn onto the various layers defined in the layers object, a key named "image" must be present whose value must be a kinetic.js image

  • undo(stage, layers, objects, all)

    Undoes lines that have been drawn.

    Parameters
    ParamTypeDetails
    stageobject

    Kinetic.js stage object where the drawing layer is

    layersobject

    Contains layers that have kinetic.js objects on them, a key named "drawing" must be present whose value must be a kinetic.js layer

    objectsobject

    Contains objects that have been drawn onto the various layers defined in the layers object, a key named "lines" must be present whose value must be an array of kinetic.js lines

    allbool

    Boolean to determine whether to undo all the line or just the previous one
