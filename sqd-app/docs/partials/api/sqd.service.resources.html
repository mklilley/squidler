resources
service in module sqd

Description

This is a service that is returned as part of resourcesProvider. Used to communicte with all squidler backend resources

Usage

resources($q, $http, $timeout, exceptions, $cordovaNetwork, $ionicPlatform);

Parameters

ParamTypeDetails
$qobject

Angular promise service

$httpobject

Angular http service

$timeoutobject

Angular timeout service

exceptionsobject

Service to catch a failed promise

$cordovaNetworkobject

ngCordova Service to tell if the network is avaiable or not

$ionicPlatformobject

Ionic platform service

Returns

object

Service object exposing methods - create, read, update

Methods

  • create(name, api, data, options)

    Creates a resource on the backend, using a POST method. This takes a minimum time set in the config of resourcesProvider using the method setWaitTime

    Parameters
    ParamTypeDetails
    namestring

    Primary key used for encoding the JSON data sent to backend and for reading the response data, e.g. 'squidles'

    apistring

    URL of the backend api for the resource described by 'name', e.g. '/api/v1/squidles'

    dataobject

    Data object specific to the backend resource, see http://#/api/backend_resource_list

    options
    (optional)
    		object

    {formData:bool}

    Returns
    promise

    Resolves to an object specific to the backend resource if the resource was created successfully. Otherwise the promise is rejected with an object of the form {message:string, code:int}

    Example

    resources.create('squidle','http://squidler.com/api/v1/squidles', {...squidle data...})

  • read(name, api, id, params)

    Reads a resource on the backend, using a GET method. This takes a minimum time set in the config of resourcesProvider using the method setWaitTime

    Parameters
    ParamTypeDetails
    namestring

    Primary key used for reading the response data from the backend, e.g. 'squidles'

    apistring

    URL of the backend api for the resource described by 'name', e.g. '/api/v1/squidles'

    idstring

    Used to uniquely identify the requested resource, e.g. '8KLv694'

    params
    (optional)
    		object

    Extra information specific to the backend resource, e.g. {guess:blue} for trying to get the Squidle prize, see http://#/api/backend_resource_list

    Returns
    promise

    Resolves to an object specific to the backend resource if the resource was read successfully. Otherwise the promise is rejected with an object of the form {message:string, code:int}

    Example

    resources.read('history', 'http://squidler.com/api/v1/history', 'joblogs')

  • update(name, api, id, data)

    Updates a resource on the backend, using a POST method wth parameter _method = put. This takes a minimum time set in the config of resourcesProvider using the method setWaitTime

    Parameters
    ParamTypeDetails
    namestring

    Primary key used for reading the response data from the backend, e.g. 'squidles'

    apistring

    URL of the backend api for the resource described by 'name', e.g. '/api/v1/squidles'

    idstring

    Used to uniquely identify the requested resource, e.g. '8KLv694'

    dataobject

    Data object specific to the backend resource, see http://#/api/backend_resource_list

    Returns
    promise

    Resolves to an object specific to the backend resource if the resource was updated successfully. Otherwise the promise is rejected with an object of the form {message:string, code:int}

    Example

    resources.update('profile', 'http://squidler.com/api/v1/profiles', 'joblogs', {...profile data...})
