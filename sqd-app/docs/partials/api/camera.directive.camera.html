camera
directive in module camera

Description

Creates an interfae to take photos and edit them

Usage

as element:
<camera
       id="{string}"
       image="{string}"
       close="{function}"
       active="{bool}">
</camera>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
idstring

Unique id for this camera element

imagestring

datURI representing a saved image

closefunction

a function to be called from the parent scope that allows the camera modal to be closed.

activebool

A boolean to determine if this camera is currently active
