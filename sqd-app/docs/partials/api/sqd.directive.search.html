search
directive in module sqd

Description

Creates an interfae to searach images from the internet

Usage

as element:
<search
       id="{string}"
       select="{function}"
       hide="{function}">
</search>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
idstring

Unique id for this search element

selectfunction

reference to a funcion on the parent scope that handles an image selected from the search list. Takes arguments (id,img), where img is an object of the form { thumb: [thumbnail link], url: [full image link], from: 'link',type: 'image'}

hidefunction

reference to a function on the parents scope to be called to hide the modal view
