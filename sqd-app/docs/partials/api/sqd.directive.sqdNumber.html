sqdNumber
directive in module sqd

Description

Creates an number input with arrows above and below to increment

Usage

as element:
<sqd-number
       min="{int}"
       max="{int}"
       step="{int}"
       disabled
       model="{int}">
</sqd-number>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
minint

Min value

maxint

Max value

stepint

Step size to increment number

disabledbool

To tell the input that it cannot be changed

modelint

value of the model to go into ng-model on the input
