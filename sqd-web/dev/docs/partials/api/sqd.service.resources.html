resources
service in module sqd

Description

This is a service that is returned as part of resourcesProvider. Used to communicte with all squidler backend resources

Usage

resources($q, $http, $timeout, exceptions, $cordovaNetwork, $ionicPlatform);

Parameters

ParamTypeDetails
$qobject

Angular promise service

$httpobject

Angular http service

$timeoutobject

Angular timeout service

exceptionsobject

Service to catch a failed promise

$cordovaNetworkobject

ngCordova Service to tell if the network is avaiable or not

$ionicPlatformobject

Ionic platform service

Returns

object

Service object exposing methods - create, read, update

Methods

  • action(name, action, id, api, data, options)

    Performs a custom action on the backend, using a POST method. This takes a minimum time set in the config of resourcesProvider using the method setWaitTime

    Parameters
    ParamTypeDetails
    namestring

    Primary key used for encoding the JSON data sent to backend and for reading the response data, e.g. 'squidle'

    actionstring

    Label for the action you want to perform e.g. "solve"

    idstring

    Used to uniquely identify the requested resource that you want to perform the action on, e.g. '8KLv694'

    apistring

    URL of the backend api for the resource described by 'name', e.g. '/api/1/squidles'

    dataobject

    Data object specific to the backend resource, see http://#/api/backend_resource_list

    options
    (optional)
    		object

    {formData:bool,silent:bool} formData is when payload is form not just JSON. silent false means any exceptions will not be displayed to the user via a popup (default is true)

    Returns
    promise

    Resolves to an object specific to the backend resource if the resource was created successfully. Otherwise the promise is rejected with an object of the form {message:string, code:int}

    Example

    resources.action('squidle', 'solve', 'http://squidler.com/api/1/squidles', {...squidle data...})
