auth
service in module sqd

Description

This is a service that is returned as part of authProvider. Used to login and verify users

Usage

auth($q, resources, stores, profiles, faqs, exceptions, $cordovaNetwork, $ionicPlatform, $rootScope);

Parameters

ParamTypeDetails
$qobject

Angular promise service

resourcesobject

Service to communicate with backend resources

storesobject

Service used to interface with the html5 local storage

profilesobject

Service used to retrieve and update a user's profile

faqsobject

Service used to retrieve the most up to date FAQs

exceptionsobject

Service to catch a failed promise

$cordovaNetworkobject

ngCordova service to check for network connectivity

$ionicPlatformobject

ionic platform service

$rootScopeobject

angularJS $rootScope service

Returns

object

Service object exposing methods - login, verify

Methods

  • login(email)

    Logs in a user

    Parameters
    ParamTypeDetails
    emailstring

    Email of the user you want to login

    Returns
    promise

    Resolves if the user was successfully logged in, rejects otherwise
