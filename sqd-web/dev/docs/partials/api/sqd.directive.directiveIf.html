directiveIf
directive in module sqd

Description

The "directiveIf" directive allows other directives to be dynamically added or removed from this element.

Usage

as attribute
<ANY directive-if
     directive-if="{object}">
   ...
</ANY>

Parameters

ParamTypeDetails
directive-ifobject

Any number of directives can be controlled with the object passed in the "directive-if" attribute on this element, e.g. {'myDirective': 'expression'}. If expression evaluates to false then attributeName will be removed from this element, otherwise it will be added
